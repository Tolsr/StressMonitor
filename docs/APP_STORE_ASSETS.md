# App Store 素材清单

本文档列出发布到 App Store 需要准备的所有素材。

## 应用图标 (AppIcon.icns)

### 必需尺寸

在 `Assets.xcassets/AppIcon.appiconset/` 中需要以下图标：

| 文件名 | 尺寸 | 比例 | 用途 |
|-------|------|------|------|
| icon_16x16.png | 16x16 | 1x | macOS |
| icon_16x16@2x.png | 32x32 | 2x | macOS Retina |
| icon_32x32.png | 32x32 | 1x | macOS |
| icon_32x32@2x.png | 64x64 | 2x | macOS Retina |
| icon_128x128.png | 128x128 | 1x | macOS |
| icon_128x128@2x.png | 256x256 | 2x | macOS Retina |
| icon_256x256.png | 256x256 | 1x | macOS |
| icon_256x256@2x.png | 512x512 | 2x | macOS Retina |
| icon_512x512.png | 512x512 | 1x | macOS |
| icon_512x512@2x.png | 1024x1024 | 2x | App Store |

### 设计建议

- 使用简洁的设计，在小尺寸下仍然可辨识
- 可以使用表情符号作为图标元素（如 😊）
- 背景建议使用渐变或纯色
- 参考 macOS Big Sur 风格（圆角方形）

## 截图

### macOS 截图尺寸

| 设备 | 尺寸（像素） | 数量要求 |
|------|-------------|---------|
| MacBook Pro 16" | 3456 x 2234 | 至少 1 张 |
| MacBook Pro 14" | 3024 x 1964 | 可选 |
| iMac 27" | 2880 x 1800 | 可选 |

### 截图内容建议

1. **主界面** - 展示状态栏图标和弹出面板
2. **应用列表** - 展示工作/娱乐应用分类
3. **自定义分类** - 展示如何修改应用分类
4. **不同压力等级** - 展示各种表情状态
5. **设置界面** - 如有设置功能

### 截取截图命令

```bash
# 使用 screencapture 命令截取
screencapture -i ~/Desktop/screenshot.png

# 或使用快捷键
# ⌘ + ⇧ + 4 选择区域截图
# ⌘ + ⇧ + 3 全屏截图
```
