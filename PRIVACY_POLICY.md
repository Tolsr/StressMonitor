# 隐私政策

最后更新日期：2026年3月23日

## 概述

压力监控（Stress Monitor）是一款 macOS 状态栏应用，旨在帮助用户监控工作与娱乐时间的平衡。我们非常重视您的隐私，本隐私政策说明我们如何收集、使用和保护您的信息。

## 我们收集的信息

### 应用使用数据

为了提供核心功能，本应用会收集以下信息：

- **当前活跃应用名称** - 用于判断您正在使用的应用
- **应用 Bundle ID** - 用于唯一标识应用程序
- **应用使用时长** - 累计各应用的使用时间

### 我们不收集的信息

- 我们**不会**收集您在应用内的具体操作
- 我们**不会**收集任何文件内容
- 我们**不会**收集任何个人身份信息
- 我们**不会**收集任何网络活动或浏览历史

## 数据存储

所有数据均存储在您的本地设备上，使用 macOS 的 UserDefaults 机制。具体包括：

- `customCategories` - 用户自定义的应用分类
- `appUsageRecords` - 应用使用记录
- `workTime` / `entertainmentTime` - 今日统计时间
- `stressReliefClicks` - 减压点击次数

**重要提示**：我们不会将任何数据上传到任何服务器或第三方服务。

## 数据保留

- 统计数据每天零点自动重置
- 用户自定义分类设置会持久保存
- 卸载应用后，所有数据将被删除

## 第三方服务

本应用**不使用**任何第三方分析、广告或追踪服务。

## 您的权利

您可以随时：

- 通过应用界面查看和修改应用分类
- 删除应用以清除所有本地数据
- 选择不使用本应用

## 儿童隐私

本应用不针对 13 岁以下儿童设计，我们不会故意收集儿童的个人信息。

## 隐私政策变更

我们可能会不时更新本隐私政策。更新后的政策将在应用更新时通知您。

# Privacy Policy

Last Updated: March 23, 2026

## Overview

Stress Monitor is a macOS menu bar application designed to help users monitor the balance between work and entertainment time. We take your privacy seriously, and this privacy policy explains how we collect, use, and protect your information.

## Information We Collect

### Application Usage Data

To provide core functionality, this application collects the following information:

- **Current active application name** - To determine which application you are using
- **Application Bundle ID** - To uniquely identify applications
- **Application usage duration** - To accumulate usage time for each application

### Information We Do NOT Collect

- We do **NOT** collect your specific actions within applications
- We do **NOT** collect any file contents
- We do **NOT** collect any personally identifiable information
- We do **NOT** collect any network activity or browsing history

## Data Storage

All data is stored locally on your device using macOS's UserDefaults mechanism. This includes:

- `customCategories` - User-defined application categories
- `appUsageRecords` - Application usage records
- `workTime` / `entertainmentTime` - Daily time statistics
- `stressReliefClicks` - Stress relief click count

**Important**: We do NOT upload any data to any server or third-party service.

## Data Retention

- Statistics are automatically reset at midnight each day
- User-defined category settings are persistently saved
- All data is deleted when the application is uninstalled

## Third-Party Services

This application does **NOT** use any third-party analytics, advertising, or tracking services.

## Your Rights

You can at any time:

- View and modify application categories through the app interface
- Delete the application to clear all local data
- Choose not to use this application

## Children's Privacy

This application is not designed for children under 13, and we do not knowingly collect personal information from children.

## Changes to This Privacy Policy

We may update this privacy policy from time to time. Updated policies will be notified when the application is updated.
